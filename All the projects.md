# All of my projects 🗃


## Github Plus ➕
- Github Plus is a projects contaning multiple userscripts (usable with a script loader) that load scripts for github and/or gist.github.
- This project aims to improve your github experience and feeling so you won't be bothered by anything.

#### Code Folding <img src="https://raw.githubusercontent.com/TheNolle/Github-Plus/master/Code%20Folding/logo.png" alt="logo" width="24" height="24"/>
- This userscript makes it so when you watch a code on Github *(javascript, css, html, etc.)* you'll be able to fold the lines so you can navigate easily between functions.

#### Issue Counter <img src="https://raw.githubusercontent.com/TheNolle/Github-Plus/master/Issue%20Counter/logo.png" alt="logo" width="24" height="24"/>
- This userscript adds a little sentence under the Repositories names showing the total of issues *(still opened, closed, etc.)* on the repository.

#### Latest Issues <img src="https://raw.githubusercontent.com/TheNolle/Github-Plus/master/Latest%20Issues/logo.png" alt="logo" width="24" height="24"/>
- This userscript adds a small button next to the "**Insights**" button wich will show you all the latest issues on the actual repository *(if there is one)*.

#### Preview files <img src="https://raw.githubusercontent.com/TheNolle/Github-Plus/master/Preview%20files/logo.png" alt="logo" width="24" height="24"/>
- This userscript adds two small buttons next to the "**Download Code**" button, one will show you the files in a basic list *(with preview if possible)*, while the second on will show it in a wider list.

#### Reveal Header <img src="https://raw.githubusercontent.com/TheNolle/Github-Plus/master/Reveal%20Header/logo.png" alt="logo" width="24" height="24"/>
- This userscript simply make it so when you scroll down on a github page, if you move your mouse at the top of your browser *(on the page)*, It'll show the header *(and will hide it once you move back your mouse)*.

#### Show Whitespaces <img src="https://raw.githubusercontent.com/TheNolle/Github-Plus/master/Show%20Whitespaces/logo.png" alt="logo" width="24" height="24"/>
- This userscript adds a little button next to the "**raw**" button *(on its left)* wich will show you every spaces on the file *(Configurable when clicking on the **TamperMonkey** logo)*.

#### Time Formatting <img src="https://raw.githubusercontent.com/TheNolle/Github-Plus/master/Time%20Formatting/logo.png" alt="logo" width="24" height="24"/>
- This userscript makes it so you can change the "**Modified __X__ ago**" by your own time formatting *(as long as it fits with [**MomentJS**](https://momentjs.com/docs/#/displaying/format/) formatting rules)*.


## Spotify Pack 📦
- Spotify Pack is a pack of two userscripts that will enhance your experience on Spotify.com by unlocking almost every Premium privileges on your free account.

#### Spotify Genius Lyrics 🎼
- This userscript displays Lyrics from Genius.com on the right side of Spotify.com *(Will work only with the web app)*.

#### Spotify AdBlocker 🚫
- This userscript skips Spotify.com's ads for you before it plays *(Will work only with the web app)*.


## Deezer Premium 🔼
- Unlocks almost every Deezer.com functions without ever paying for it *(Will work only with the web app)*.
